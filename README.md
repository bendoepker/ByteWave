This is the ByteWave DAW which is a lightweight digital audio workstation

Libraries / API's:
  Qt Library
    Creates the GUI
    Handles Keyboard/Mouse Events
    Terminates the program
  PortAudio
    Handles data buffers being sent to outputs
    Fetches data buffers from inputs
    Handles ASIO configurations

Building the Project:
  This project requires the Qt Framework (V6.7.2) to be installed prior to compilation
  
  Contained within the repository is a Visual Studio Solution which can be used to compile and run the application
